# 北京薪资个税计算器 (2020-2026)

一个专业且详尽的北京地区薪资个税计算器，基于 Vue 3 + Vite 开发。它不仅能计算个人税后所得，还能帮助你了解企业的用工成本。

## 🌟 项目亮点

- **多年度支持**：涵盖 2020 年至 2026 年的北京社保及公积金政策标准。
- **政策精准匹配**：严格遵循北京“社保年度”（每年 7 月至次年 6 月）的基数调整逻辑。
- **年终奖计算**：支持年终奖“单独计税”与“合并计税”两种方式的对比。
- **专项附加扣除**：完整涵盖子女教育、赡养老人、住房配套等 7 大专项附加扣除。
- **数据可视化**：采用 ECharts 实时展示薪资组成、企业成本与税后所得的比例。
- **现代美学设计**：基于 Glassmorphism（磨砂玻璃）风格，提供深色模式友好的高端视觉体验。

## 🛠️ 技术栈

- **框架**: [Vue 3 (Composition API)](https://vuejs.org/)
- **逻辑**: [VueUse](https://vueuse.org/) (状态管理与实用工具)
- **图表**: [Apache ECharts](https://echarts.apache.org/) (数据可视化)
- **构建**: [Vite](https://vitejs.dev/)
- **样式**: CSS (Glassmorphism 风格) + Sass

## 📖 使用文档

### 1. 基础配置
- **选择年份**：计算器会自动加载该年份对应的北京社保上下限及平均工资。
- **缴纳公积金**：勾选后可输入公积金基数。
- **统一标准**：如果全年的薪资和基数一致，可以在“全年统一标准”中输入；如果上半年和下半年政策不同（如 7 月基数调整），可分别设置。

### 2. 社保与个税逻辑
- **社保年度**：北京社保基数通常在每年 7 月调整。本计算器将 1-6 月归为上一政策年度，7-12 月归为当前政策年度。
- **累计计税**：个税计算采用专项扣除后的累计年度预扣预缴法，能精准还原每月个税的变化过程。
- **专项扣除**：用户可勾选多项专项附加扣除，金额按国家法定标准自动预填。

### 3. 数据解读
- **年度概览**：快速查看年度总税前、总税后、个人缴纳总计及企业总支出。
- **月度明细**：表格形式列出每月具体的个税税率、专项扣除及到手月薪。
- **比例图表**：清晰展示你的薪水流向。
